# OOP
Getting employee details using OOP

## Installation
1.  Git clone this repository `https://github.com/Ruqoyah/OOP.git`
2.  Change your directory `cd OOP`
3.  Install all dev dependencies `npm install`
4.  Run test `npm run test`
